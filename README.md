# Economic Dashboard
This repository contains an interactive data visualization dashboard built with Streamlit to explore key Indonesian macroeconomic indicators, including GDP, inflation, trade, poverty, employment, and human development.
The dashboard transforms official statistics into clear, visual, and actionable insights to support analysis and decision-making.

## Data Sources
- **Badan Pusat Statistik (BPS) Indonesia**
- Data are presented in aggregated and visualization-friendly formats
- Some sections use sample or simulated data for demonstration purposes
